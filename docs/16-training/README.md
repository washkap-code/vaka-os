# VAKA Training and User Guidance

This directory contains controlled user and operator guidance. Guidance must
describe current behaviour accurately, distinguish planned capabilities, and
link consequential operations to their approval, audit and rollback controls.

## Current guides

- [Super Admin User Guide](SUPER-ADMIN-USER-GUIDE.md) — the single source bundled into the authenticated Platform Administration console.

Product behavior and architecture remain governed by the Constitution,
accepted ADRs, authoritative domain architecture and approved Mission Packs.
