# VAKA Finance documentation

## Authority

The authoritative target architecture is [Book Eight - VAKA Finance & Accounting Intelligence Architecture](../06-master-programme-blueprint/books/08-finance-and-accounting/README.md).

Files in this directory provide current-state audits, write-path maps, invariant definitions, risks, migration plans, verification and remediation evidence. They may refine implementation evidence but may not weaken the authoritative architecture, Constitution or repository instructions.

## Current-state evidence sequence

Start with `00-current-state-audit.md`, `01-financial-write-paths.md`, `02-schema-gap-analysis.md`, `03-ledger-invariants.md`, `05-risk-register.md`, and the numbered verification/remediation evidence. Material accounting behavior changes require an approved finance ADR/Mission Pack and applicable professional review.
