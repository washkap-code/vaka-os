# VAKA Knowledge System (VKS)

## Purpose

The VKS is VAKA's governed control plane. It owns authority, indexes, canonical terminology, architecture decisions, programme status, traceability and knowledge lifecycle. Detailed product, engineering, finance, security, operational and blueprint specifications live in `docs/` and are linked from the VKS.

Document precedence and the control/specification split are defined by [ADR-001](18-decisions/ADR-001-document-authority.md). No implementation may contradict the Constitution, accepted ADRs, authoritative domain architecture or its approved Mission Pack.
